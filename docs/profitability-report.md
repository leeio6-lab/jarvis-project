## 10. 결론

### $4.99는 맞는 가격인가?

**YES — API 비용 관점에서는 매우 건강한 마진.**

- 현재 모델(gpt-4.1-mini) 기준 마진율 87.8%
- 헤비유저까지 감안해도 마진 67%+
- 경쟁 서비스($10-20) 대비 저가이면서도 수익성 확보

### 가격을 올려야 하나?

| 가격 | 장점 | 단점 |
|------|------|------|
| $2.99 | 진입 장벽 낮음, 빠른 사용자 확보 | 마진 $2.38 (서버비 빼면 $2.13) |
| **$4.99** | 균형점. "커피 한 잔" 심리적 기준 | |
| $9.99 | 마진 $9.38, sonnet 사용 가능 | 사용자 이탈 |
| $4.99 + $9.99 Pro | 2티어로 양쪽 커버 | 개발/운영 복잡도 |

### 추천 전략
